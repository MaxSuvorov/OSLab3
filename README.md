# Лабораторная работа: Кроссплатформенная программа с процессами

## Описание
Программа реализует многопроцессную систему с общим счетчиком и логированием. Поддерживает Windows и POSIX-совместимые системы (Linux, macOS).

## Функциональность
1. Запись в лог-файл при старте (PID и время запуска)
2. Автоматическое увеличение счетчика каждые 300 мс
3. Установка значения счетчика через консоль
4. Периодическое логирование (каждую секунду)
5. Запуск дочерних процессов (каждые 3 секунды)
6. Поддержка нескольких запущенных программ с одним мастер-процессом
